# 科普与航天科技影视中心 (Science_Video)

这是一个独立、自托管的科普与航天科技影片展示中心网站，原为 `caih1943.github.io` 主页的一个子模块，现已独立为单一仓库运行。

## 🌟 网站特色
- **现代美观的 UI 设计**：采用深色太空背景、渐变文字、毛玻璃效果和流线型网格卡片布局。
- **高稳定性的本地视频播放**：视频文件（`.mp4`）直接托管于此仓库中，点击卡片即可在弹窗内通过 HTML5 播放器流畅播放，避免了 Google Drive 第三方预览链接失效或被浏览器拦截的问题。
- **自适应响应式布局**：完美适配手机、平板及电脑屏幕。

## 📁 目录结构说明
```text
Science_Video/

```
